Comment ça marche · 3 étapes

Audit en ligne commande.

Je prépare tout · tu colles · Claude fait le reste.

01

Tu cliques "Auditer"

Sur la ligne d'un article. Je te prépare un prompt complet et je le copie dans ton presse-papier.

02

Tu colles dans Claude Code

Dans ton terminal à la racine du projet. Claude Code invoque les 2 skills content-quality-auditor + geo-content-optimizer, puis écrit le rapport dans audits/<slug>/<date>.md.

03

Tu cliques "Rafraîchir"

Je lis les rapports écrits, j'affiche les scores dans la table. Clique sur un score pour voir le rapport complet · top 5 améliorations · plan d'action.
