Les réglages
de la maison.

Ici tu retrouves toutes les clés API à configurer pour que les modules marchent. Rien de secret ne fuit ici · on ne montre que le nom de la variable d'environnement à positionner, pas la valeur.

Où mettre ces clés ? Les secrets vont dans .env.local à la racine du projet (local) et dans Vercel · Settings · Environment Variables (prod). Tu ne les commit jamais dans git · le fichier .env.local est dans .gitignore par défaut.
