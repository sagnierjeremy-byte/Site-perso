## 3 à 5 recos prioritaires

1. **Title à ≤60 chars qui place le mot-clé devant** : `"Créer un podcast avec une voix IA en 24h · Making-of"` (53 chars). Le format "Comment j'ai…" est cliquable mais bouffe 12 chars de runtime SEO.

2. **Meta description recadrée à 150-155 chars** : `"Trois épisodes de podcast narratif style Wondery sur la guerre IA, créés en 24h pour 33 dollars avec voix synthétiques ElevenLabs et sous-agents Claude."` (~152 chars).

3. **Ajouter `PodcastEpisode` JSON-LD** (un par épisode) ou `PodcastSeries` à côté du TechArticle existant. L'article HÉBERGE 3 épisodes audio R2 avec `<audio src>` — Google peut les indexer en Google Podcasts/audio search si le schema est là. Énorme manque vs le potentiel.

4. **Ajouter 5+ liens externes** : Wondery (officiel), ElevenLabs voice library, l'article OpenAI sur le board (novembre 2023), procès Musk vs OpenAI (Reuters), DeepSeek paper. Boost E-E-A-T + crédibilité GEO (les LLM citent uniquement les pages avec sources externes).

5. **Afficher la date côté lecteur** dans hero-meta : remplacer "Outils Claude Code · ElevenLabs" par "Date · 23 avril 2026" et déplacer les outils ailleurs (ou garder les deux). Le lecteur veut savoir si c'est récent (sujet IA = perissable).
