## Mot-clé principal identifié
**"open source pour débutant"** (ou "open source expliqué simplement")

Justification : l'article est un guide pédagogique pour quelqu'un qui n'a jamais codé. L'intent matche les requêtes "c'est quoi l'open source", "open source vs gratuit", "logiciels open source à essayer". Le mot-clé "open source" + qualifier "pour débutant/non-dev/expliqué" est la cible naturelle.

Mots-clés secondaires suggérés :
- licence open source MIT GPL Apache
- logiciels open source 2026
- IA open source vs propriétaire (Mistral vs ChatGPT)
- alternatives open source aux outils SaaS
