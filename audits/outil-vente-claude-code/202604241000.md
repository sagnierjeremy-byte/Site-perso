## Mot-clé principal
**"construire un outil avec Claude Code sans coder"**

Justification : l'angle making-of "non-dev qui livre un outil métier en 3 soirées" matche directement l'intent informationnel/aspirationnel de quelqu'un qui découvre Claude Code. Le pivot sémantique fort = "sans coder" / "non-dev" — totalement absent du title actuel.

Secondaires : "Claude Code outil métier", "Claude Code étude de cas", "vibe coding non-dev", "automatiser un fichier Word avec IA".

---

## 3 à 5 recos prioritaires

1. **Réécrire le title à 55-60 chars en injectant l'intent "sans coder"** : par exemple "Claude Code sans coder : j'ai construit l'outil de vente de ma femme" (62 chars, gardable). Sinon "Construire un outil avec Claude Code — sans être dev" (54 chars). Le suffixe "— par Jérémy Sagnier" doit sauter, le brand name n'apporte rien sur une SERP où Jérémy n'est pas une marque connue.

2. **Compresser la meta description à 150-158 chars** en gardant le hook "3 soirées + Netflix + chargée de clientèle" : "Ma femme, chargée de clientèle en assurance, n'est pas dev. En 3 soirées devant Netflix on a construit son outil de vente avec Claude Code." (140 chars, propre).

3. **Ajouter 4-6 liens internes contextuels** dans le corps : pointer vers `claude-code.html` (sur la 1ère mention "Claude Code"), `loops-claude.html` (section "Soirée 1" sur le déroulé itératif), `agents-ia-guide.html` (section "ce que ça veut dire pour toi"), `veille-pour-demain.html` (section "lecons" sur "construire pour soi"). Maillage actuel sous-exploité pour PageRank interne.

4. **Aligner les `datePublished` dans les 2 JSON-LD** : actuellement Article = 2026-04-15 et TechArticle = 2026-04-20. Garder une seule date de publi (15 avril) + garder dateModified à 20 avril. Sinon Google peut considérer le balisage comme cassé. Idéalement : supprimer le bloc TechArticle redondant ou le fusionner avec Article (un seul JSON-LD par page).

5. **Ajouter 2-3 liens externes vers sources tierces** pour booster E-E-A-T : page officielle Claude Code (`anthropic.com/claude-code`), définition wikipedia "Vibe coding" ou article Karpathy de référence, screenshot/repo du projet si possible. Un article making-of sans aucun lien externe sent l'isolement sémantique — Google a besoin d'ancrage à l'écosystème.

## Quick wins
- Title à 55-60 chars sur l'angle "sans coder" → +CTR estimé 10-15%
- Meta desc à 140-158 chars → fin de la troncature SERP
- Twitter:image explicite (manque la balise) — sinon Twitter affiche le 1er `<img>` du DOM par défaut, qui n'est pas optimal
- Fusionner les 2 JSON-LD en un seul (Article OU TechArticle, pas les deux) avec dates cohérentes
- 4 liens internes contextuels à injecter sans toucher au copy (juste wraper du texte existant)
