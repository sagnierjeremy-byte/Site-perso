## 3 à 5 recos prioritaires (actionnables)

1. **Compléter OG + ajouter Twitter Card + canonical** · Tu as 3 balises OG mais ni `og:image` ni `og:url` ni Twitter Card → partage cassé sur LinkedIn et X. · Ajouter `og:image` (1200×630, screenshot du tuto), `og:url`, `twitter:card=summary_large_image`, `twitter:site`, `twitter:creator`, `twitter:title`, `twitter:description` + `<link rel="canonical" href="https://jerwis.fr/articles/tuto-agent-gmail.html">`.

2. **Couper title à 60 chars max** · 99 chars = Google coupe à ~60 et perd "résumé matinal" qui est ton hook. · Proposition : "Tuto Claude Code : agent Gmail qui trie tes mails (45 min)" (59 chars) — garde tutorial intent + outil + bénéfice + temps.

3. **Réécrire meta desc à 155 chars** · 193 chars actuels → tronqué. · Proposition (153 chars) : "Tutoriel pas-à-pas : crée un agent Claude Code qui trie 80 mails par jour et t'envoie les 3 urgents chaque matin. Setup 45 min, 5 €/mois."

4. **Ajouter 4-6 liens externes critiques** · Pour un tuto, l'absence de lien vers la doc Claude Code, le repo GitHub MCP Gmail, raf.dev, Zapier, launchd Apple = friction lecteur + perte trust signal. · Liens minimum : doc Claude Code (claude.com/docs), repo GitHub MCP Gmail mentionné, raf.dev (article cité), Zapier, Apple launchd doc.

5. **Ajouter Schema.org HowTo en plus de TechArticle** · Le contenu est un how-to en 6 étapes avec prérequis, durée, coût → parfait candidat. Google peut afficher steps en rich result. · Ajouter un `<script type="application/ld+json">` avec `@type: HowTo`, `step[]` listant les 6 étapes (id `#installation`, `#connecteur`, etc.), `totalTime: PT45M`, `estimatedCost`.
