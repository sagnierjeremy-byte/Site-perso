# Adaptations Linguistiques et Culturelles — Eurofiscalis Transcréation

Ce document guide l'adaptation du style JIM dans chacune des 7 langues Eurofiscalis. Le style JIM (BLUF, encarts, gras stratégique, zéro AI-isme) s'applique dans toutes les langues, mais chaque langue a ses particularités culturelles et ses pièges de rédaction IA.

---

## Principes universels (toutes langues)

Ces règles ne changent **jamais**, quelle que soit la langue :

1. **BLUF** — chaque paragraphe commence par la conclusion
2. **Gras = chemin de lecture** — max 5-7% du texte
3. **Shortcodes** — aucun chiffre variable hardcodé
4. **Pyramide inversée** — H2 ordonnés par importance décroissante
5. **Encarts** — 3-4 "Conseil Expert" + 2-3 "Point de Vigilance" par article
6. **Listes à puces** — pour les énumérations de 3+ éléments
7. **Pas de chauffe** — l'accroche répond directement à l'intention

## Français (FR)

**Sous-domaine** : eurofiscalis.com (domaine principal) ou fr.eurofiscalis.com

**Formalité** : vouvoiement systématique. Ton expert-accessible.

**AI-ismes français à éviter** :

| Interdit | Alternative |
|----------|-------------|
| "Il est essentiel de noter que..." | Supprimer, commencer directement |
| "Dans cette section, nous verrons..." | Supprimer |
| "Pour conclure, il convient de souligner..." | Aller droit au fait |
| "Il est important de comprendre que..." | Supprimer |
| "N'hésitez pas à..." | "Contactez-nous" |
| "En effet..." (début de phrase) | Supprimer |
| "Force est de constater..." | Constater directement |
| "Dans le cadre de..." | Reformuler |
| "À ce titre..." | Supprimer |

**Spécificités** : le français fiscal utilise un mélange de termes français et d'anglicismes acceptés ("reverse charge" coexiste avec "autoliquidation"). Utiliser le terme français en priorité, avec le terme anglais entre parenthèses si c'est le terme technique de référence.

## Anglais (EN)

**Sous-domaine** : en.eurofiscalis.com

**Formalité** : ton professionnel mais direct. Pas de "Dear reader" ni de formalisme excessif. Utiliser "you" librement.

**AI-ismes anglais à éviter** :

| Interdit | Alternative |
|----------|-------------|
| "It is important to note that..." | Remove, start directly |
| "In this section, we will explore..." | Remove |
| "Let's delve into..." | Remove |
| "It's worth mentioning that..." | Remove |
| "In today's ever-changing landscape..." | Remove |
| "Navigating the complexities of..." | State the rule directly |
| "It goes without saying..." | If it goes without saying, don't say it |
| "As a matter of fact..." | Remove |
| "In conclusion..." | Direct CTA instead |

**Spécificités** : l'anglais est souvent la langue pivot (les agences envoient en anglais). Attention à distinguer UK English (HMRC, VAT) vs US English (IRS, Sales Tax) — Eurofiscalis utilise le contexte européen, donc privilégier UK English pour le vocabulaire fiscal.

## Italien (IT)

**Sous-domaine** : it.eurofiscalis.com

**Formalité** : vouvoiement de politesse ("Lei" formel). Ton expert mais chaleureux — les Italiens apprécient un registre légèrement plus personnel que le français.

**AI-ismes italiens à éviter** :

| Interdit | Alternative |
|----------|-------------|
| "È importante sottolineare che..." | Togliere, iniziare direttamente |
| "In questa sezione vedremo..." | Togliere |
| "Come abbiamo visto in precedenza..." | Togliere |
| "Vale la pena ricordare che..." | Togliere |
| "Non esitate a contattarci..." | "Contattateci" |
| "In conclusione..." | CTA diretto |

**Termes fiscaux clés** : IVA (pas TVA), Agenzia delle Entrate, partita IVA, codice fiscale, dichiarazione IVA, LIPE (comunicazione liquidazioni periodiche), spesometro, esterometro, fattura elettronica.

**Particularité** : l'Italie a la facturation électronique obligatoire (SDI/Sistema di Interscambio). C'est un sujet à intégrer systématiquement dans les articles sur la facturation et la TVA en Italie.

## Allemand (DE)

**Sous-domaine** : de.eurofiscalis.com

**Formalité** : vouvoiement systématique ("Sie" avec majuscule). L'allemand professionnel est plus formel que le français. Les phrases peuvent être plus longues et structurées — c'est acceptable en allemand, contrairement au français où on préfère les phrases courtes.

**AI-ismes allemands à éviter** :

| Interdit | Alternative |
|----------|-------------|
| "Es ist wichtig zu beachten, dass..." | Streichen, direkt anfangen |
| "In diesem Abschnitt werden wir..." | Streichen |
| "Zusammenfassend lässt sich sagen..." | Direkt zum CTA |
| "Es sei darauf hingewiesen, dass..." | Streichen |
| "Zögern Sie nicht..." | "Kontaktieren Sie uns" |
| "Im Folgenden..." | Streichen |

**Termes fiscaux clés** : Mehrwertsteuer (MwSt), Umsatzsteuer (USt), Finanzamt, Steuernummer, USt-IdNr (Umsatzsteuer-Identifikationsnummer), Voranmeldung, Zusammenfassende Meldung, Reverse-Charge-Verfahren.

**Particularité** : les noms composés allemands sont longs (Umsatzsteuervoranmeldung). Les utiliser correctement — ne pas les décomposer artificiellement. Les abréviations sont courantes et attendues (MwSt, USt, ZM).

## Polonais (PL)

**Sous-domaine** : pl.eurofiscalis.com

**Formalité** : vouvoiement de politesse ("Pan/Pani" ou forme "Państwo" pour le pluriel). Le polonais professionnel est assez formel.

**AI-ismes polonais à éviter** :

| Interdit | Alternative |
|----------|-------------|
| "Warto zauważyć, że..." | Usunąć, zacząć bezpośrednio |
| "W tej sekcji omówimy..." | Usunąć |
| "Podsumowując..." | Bezpośredni CTA |
| "Należy podkreślić, że..." | Usunąć |
| "Nie wahaj się..." | "Skontaktuj się z nami" |

**Termes fiscaux clés** : VAT (utilisé tel quel en polonais), podatek od towarów i usług (PTU — terme officiel), Urząd Skarbowy, NIP (Numer Identyfikacji Podatkowej), JPK (Jednolity Plik Kontrolny), deklaracja VAT-7, INTRASTAT.

**Particularité** : la Pologne a le système JPK (SAF-T polonais) qui est très spécifique. Les déclarations sont liées au fichier JPK_VAT. Sujet incontournable pour tout article sur la TVA en Pologne.

## Tchèque (CS)

**Sous-domaine** : cs.eurofiscalis.com

**Formalité** : vouvoiement ("Vy" avec majuscule dans la correspondance formelle). Ton professionnel mais accessible.

**AI-ismes tchèques à éviter** :

| Interdit | Alternative |
|----------|-------------|
| "Je důležité poznamenat, že..." | Odstranit, začít přímo |
| "V této části se podíváme na..." | Odstranit |
| "Závěrem lze říci..." | Přímý CTA |
| "Stojí za zmínku, že..." | Odstranit |
| "Neváhejte nás kontaktovat..." | "Kontaktujte nás" |

**Termes fiscaux clés** : DPH (daň z přidané hodnoty), Finanční správa, DIČ (daňové identifikační číslo), kontrolní hlášení (rapport de contrôle — spécificité tchèque), souhrnné hlášení, přiznání k DPH.

**Particularité** : la Tchéquie a le "kontrolní hlášení" (rapport de contrôle TVA) qui est unique en Europe — obligation supplémentaire en plus de la déclaration TVA classique. Sujet spécifique à mentionner systématiquement.

## Norvégien (NO)

**Sous-domaine** : no.eurofiscalis.com

**Formalité** : tutoiement acceptable en norvégien professionnel (la Norvège est une culture égalitaire). On peut utiliser "du" ou rester impersonnel. Ton direct et pragmatique — très compatible avec le style JIM naturellement.

**Note linguistique** : le norvégien a deux formes écrites — bokmål et nynorsk. **Utiliser le bokmål** (forme la plus courante, utilisée par ~85% de la population et la majorité des entreprises).

**AI-ismes norvégiens à éviter** :

| Interdit | Alternative |
|----------|-------------|
| "Det er viktig å merke seg at..." | Fjern, start direkte |
| "I denne delen skal vi se på..." | Fjern |
| "Oppsummert kan vi si at..." | Direkte CTA |
| "Det er verdt å nevne at..." | Fjern |
| "Ikke nøl med å kontakte oss..." | "Kontakt oss" |

**Termes fiscaux clés** : MVA (merverdiavgift), Skatteetaten, organisasjonsnummer, MVA-registeret, omsetningsoppgave, SAF-T (format norvégien), skattemelding.

**Particularité** : la Norvège n'est PAS membre de l'UE (mais fait partie de l'EEE). Les règles TVA diffèrent significativement — pas de directives EU applicables directement, mais un régime fiscal propre. Attention à ne pas appliquer les règles EU par défaut aux articles sur la Norvège.
