01
— L'outil central
Claude Code Max 200 $/mois

Le cerveau de toute ma stack. Claude Code, c'est Anthropic dans ton terminal. Il lit tes fichiers, il écrit du code, il lance des commandes, il corrige ses erreurs. Je l'utilise 6 à 8 heures par jour depuis six mois · pas exagéré, c'est mesuré.

Pourquoi lui
La version Max (le plan à 200 $) donne un accès prioritaire aux modèles Opus et Sonnet, avec des limites de tokens qu'un usage intensif ne touche jamais. Pour un entrepreneur qui l'utilise tous les jours, c'est rentabilisé en 48 h d'économies de temps.
Installation
Télécharger l'app sur claude.com/download. Double-clic, connexion avec ton compte Anthropic, go.
Mon usage
Rédaction d'articles, maintenance de ce site, scripts d'automatisation (newsletter, brainstorm, social-poster), debugging, recherche, audit SEO. Tout y passe.

02
— Le chaînon manquant
dev-browser Gratuit

Un navigateur sans fenêtre que ton IA pilote en écrivant un petit script. Tu demandes à Claude d'aller sur une page, de la scraper, de la tester, de la screenshoter · il le fait. Sans que tu ouvres Chrome à la main, sans que tu écrives le code.

Pourquoi lui
Claude connaît énormément de pages via son entraînement, mais ne connaît pas les pages récentes et ne peut pas interagir. Dev-browser ouvre une vraie instance Chromium en arrière-plan, Claude y envoie des ordres. Résultat : tu peux demander à Claude « vérifie que ma page est à jour » ou « remplis ce formulaire et dis-moi ce qui se passe ». Magique.
Installation
Demande à Claude Code « installe dev-browser pour moi ». Il te sortira la commande à jour (généralement via npm install -g).
Mon usage
Vérifier après chaque déploiement que le site répond bien · scraper une source pour un article · tester le formulaire newsletter en local · prendre des screenshots pendant les refontes UI. Je l'appelle 5 à 10 fois par jour via Claude.
Mon article complet → Pré-requis · Claude Code
03
— L'hébergement sans friction
Vercel Gratuit pour toi

Tu push du code sur GitHub, Vercel le déploie tout seul en 30 à 45 secondes. Plus besoin de serveur, plus besoin de savoir ce que c'est qu'un serveur. Mon site entier tourne chez Vercel, gratuitement. Toi aussi tu peux.

Pourquoi lui
Le plan Hobby (gratuit) suffit pour un site perso, un blog, une landing, une newsletter. Bande passante large, déploiements illimités, domaine personnalisé possible. Pas de carte bancaire demandée tant que tu ne dépasses pas. Parfait pour démarrer.
Installation
Crée un compte sur vercel.com (gratuit, 30 secondes), connecte ton compte GitHub, clique sur « Import Project ». Vercel détecte la techno, build, déploie. Fin.
Mon usage
Je déploie ce site à chaque commit git. Mon endpoint newsletter (/api/subscribe.js) tourne en tant que fonction serverless Vercel. Je n'ai jamais eu à intervenir sur l'infra. Zéro maintenance.

04
— L'envoi d'emails pour devs
Resend Gratuit jusqu'à 3k/mois

Le service que j'utilise pour envoyer mes newsletters et gérer mes inscrits. API ultra simple, interface propre, logs détaillés, pas d'usine à gaz type Mailchimp. 3 000 emails gratuits par mois, largement suffisant pour commencer.

Pourquoi lui
Resend est pensé pour les devs (et donc pour nos IA qui codent pour nous). Une clé API, 3 lignes pour envoyer un email, un éditeur d'audience propre, statistiques claires. Aucun gimmick marketing, zéro upsell. Le plan gratuit est vraiment utilisable · pas un leurre comme la plupart des concurrents.
Installation
Compte sur resend.com, vérification de ton domaine d'envoi (DNS simple), clé API générée. Tu donnes la clé à Claude, il écrit le code d'intégration pour toi.
Mon usage
J'ai mon audience « Vendeurs Amazon » (119 contacts) hébergée chez Resend. Mon endpoint Vercel /api/subscribe ajoute chaque inscrit automatiquement. J'envoie ma newsletter du vendredi depuis l'interface web.

05
— L'automatisation no-code
n8n Gratuit self-hosted · 20 $/mois cloud

L'équivalent de Zapier ou Make, mais open-source et auto-hébergeable. Tu relies des services entre eux en glissant-déposant des blocs. Parfait quand tu veux que ton IA déclenche des actions en cascade sur 3 ou 4 outils sans écrire de code.

Pourquoi lui
Contrairement à Zapier qui facture chaque action ou Make qui facture chaque opération, n8n self-hosted est gratuit à usage illimité. Tu paies juste ton serveur (5 $/mois chez Hetzner ou gratuit sur Railway). Et si tu ne veux pas héberger, n8n Cloud à 20 $/mois fait le job sans limitation.
Installation
Option facile : compte sur n8n.io, plan cloud à 20 $. Option économique : un VPS à 5 $/mois + demande à Claude « installe n8n sur mon VPS ». Il te tient la main.
Mon usage
Automatiser des workflows qui touchent plusieurs services (Gmail → CRM → Slack → Notion). Je n'y suis pas encore à plein régime — je préfère mes propres scripts Claude Code — mais n8n brille dès que tu veux brancher 4-5 outils tiers ensemble.

06
— Le terminal qui ne te dégoûte pas
Ghostty Gratuit

Un terminal moderne, open-source, hyper fluide. C'est dans Ghostty que je lance Claude Code tous les jours. Comparé au Terminal Apple par défaut (qui fonctionne, mais qui date de 2005), c'est le jour et la nuit.

Pourquoi lui
Le Terminal Apple est moche mais fonctionnel. Ghostty est tout aussi simple, mais avec des couleurs claires, un rendu vectoriel des polices, un comportement moderne. Zéro configuration à la main · tu ouvres, tu utilises. Pour quelqu'un qui passe 8 heures par jour dans un terminal, ça change tout.
Installation
Télécharger sur ghostty.org (Mac uniquement pour l'instant · Linux arrive). Double-clic, glisser dans Applications, ouvrir. C'est tout.
Alternative
Si tu veux une version avec IA intégrée directement dans le terminal, Warp (payant, 20 $/mois) est plus riche. Mais pour un non-dev qui lance juste Claude Code dedans, Ghostty suffit largement et est gratuit.
Télécharger Ghostty ↗ Alternative · Warp ↗
07
— La base de données sans serveur à gérer
Supabase Gratuit < 500 Mo

Dès que ton site, ton formulaire ou ton agent a besoin de stocker des données (utilisateurs, réponses, leads, mémoire longue), il te faut une base de données. Supabase est la plus simple du marché pour un non-dev · une interface web propre, un plan gratuit sérieux, zéro serveur à administrer.

Pourquoi lui
Contrairement à MongoDB, MySQL ou PostgreSQL que tu dois héberger toi-même, Supabase te donne une base PostgreSQL managée dans le cloud avec authentification, stockage de fichiers et fonctions serverless · tout dans le même dashboard. Le plan gratuit tient 500 Mo de données et 50 000 utilisateurs connectés par mois · largement de quoi faire tourner un premier projet ou un MVP.
Installation

1. Compte gratuit sur supabase.com (GitHub ou email, 20 secondes).

2. Crée un projet · choisis une région proche de toi (Paris si possible).

3. Récupère ton URL + clé API dans Project Settings → API.

4. Donne tout ça à Claude Code · « connecte ce projet à Supabase, créée une table inscrits avec email et date ». Il installe la bibliothèque et écrit le code.

Mon usage
Dans les projets Eurofiscalis, Supabase stocke les formulaires de qualification, les leads entrants, les sessions d'agents IA. Pour mon site perso, je n'en ai pas encore besoin (tout est statique), mais dès que j'aurai un système de commentaires, une base de feedbacks ou une mémoire longue pour un agent · ça passera par Supabase.
Pour qui c'est
Dès que ton site passe de « page vitrine » à « application qui retient quelque chose ». Formulaire qui sauve des réponses, newsletter avec segments, agent qui se souvient des conversations passées · Supabase est le chaînon de stockage.

— Partie 2 · Stack avancée

4 outils pour aller plus loin.

Ces 4 outils ne sont pas dans la stack de base · tu n'en as pas besoin tant que tu n'as pas un cas d'usage précis qui les justifie. Je te les présente quand même, parce que je les utilise tous pour des projets concrets, et parce que tu les croiseras forcément si tu creuses dans l'IA ou le contenu.

Sanity si tu veux un vrai CMS pour un blog multi-langues. Zernio si tu publies régulièrement sur les réseaux. fal.ai si tu génères des images ou vidéos par IA. Remotion si tu veux automatiser des vidéos programmatiques.

Ma règle · je n'installe jamais un outil du tier 2 avant d'avoir un projet qui a buté sur une limite du tier 1. Si tu as juste envie de tester, fais-le ailleurs que dans ta stack de prod.

08
— Le CMS headless
Sanity Gratuit < 3 users

Un vrai CMS pour gérer du contenu structuré · articles, pages, catégories, auteurs. L'interface est propre, l'API est excellente, et tu gardes la main sur la présentation (Sanity ne force aucun design). Je l'utilise pour le blog d'Eurofiscalis · 7 langues, 14 catégories, articles traduits via une seule référence métadata.

Quand l'ajouter
Quand tu dépasses ~20 articles sur ton site et que tu commences à te dire *« j'aimerais une interface pour gérer ça sans toucher au code »*. Ou si tu as du multi-langues sérieux (3+ langues). En dessous, un simple dossier articles/ en Markdown suffit.
Démarrage
Compte sur sanity.io. Crée un projet via leur CLI (npm create sanity@latest). Claude Code te tient la main sur le schéma de données. Plan gratuit tient pour 3 utilisateurs + 10 000 documents.
Mon usage
Blog Eurofiscalis · 7 articles Belgique en draft (6 traductions + 1 original), taxonomies publiées, pipeline MD → Sanity reproductible. Pas encore utilisé pour ce site perso · trop léger pour le justifier.

09
— Le planificateur social
Zernio 20 à 50 €/mois

Un SaaS français pour planifier et publier sur LinkedIn, X, Instagram, Threads depuis une seule interface. API propre (j'en ai fait une intégration dans mon back-office), statistiques suffisantes, pricing fair. Alternatif à Buffer ou Hootsuite, moins cher et plus simple.

Quand l'ajouter
Quand tu publies régulièrement sur 2+ réseaux et que tu passes plus de 30 minutes par semaine à le faire à la main. En dessous, Claude + copy-paste manuel suffisent.
Démarrage
Compte sur zernio.com. Connecte tes réseaux, plan à 20 €/mois pour 2 réseaux, 50 €/mois pour les 4. API disponible avec une clé Bearer · je l'ai intégrée à mon back-office pour publier depuis mes drafts.
Mon usage
Mon back-office génère des drafts LinkedIn et X à partir de chaque article publié via un bouton "Social-poster". Je valide, je programme dans Zernio, la suite est automatique.

10
— L'API IA pour visuels
fal.ai Pay-as-you-go

Une plateforme qui expose des dizaines de modèles IA pour générer images, vidéos, voix, animations · tous derrière une seule API unifiée. Infiniment plus simple que d'assembler Replicate, RunPod ou HuggingFace à la main. Tarification à l'usage, quelques cents par génération selon le modèle.

Quand l'ajouter
Quand tu veux intégrer de la génération d'images ou vidéos IA dans un workflow automatisé. Pour un usage ponctuel (une image de blog par semaine), Midjourney direct ou ChatGPT image suffit.
Démarrage
Compte sur fal.ai, 10 $ de crédits offerts. Clé API dans le dashboard. Claude Code sait appeler l'API directement · tu lui demandes « génère une image Flux avec ce prompt », il code l'appel.
Mon usage
Dans le projet Leo Reels Pipeline (prévu pour la newsletter Eurofiscalis) · génération d'avatars vidéo via Hedra + voix via ElevenLabs, assemblés par Remotion. fal.ai orchestre une partie du pipeline images.

11
— Les vidéos programmatiques
Remotion Gratuit (usage solo)

Écris une vidéo en code (en fait en React) · textes, images, animations, transitions. Remotion exporte la vidéo finale au format MP4. La seule manière sérieuse de générer 50 variantes d'un reel en un script. Gratuit pour usage personnel, payant si tu l'utilises dans un produit commercial à plus de 3 développeurs.

Quand l'ajouter
Quand tu veux industrialiser la production de vidéos courtes (reels IG, TikTok) avec du contenu dynamique · titre, sous-titres, B-roll changeant à chaque version. Pour une vidéo ponctuelle, CapCut ou Descript suffisent largement.
Démarrage
npx create-video dans ton terminal crée un projet. Tu écris ta vidéo en JSX, tu lances npm start pour prévisualiser, npm run build pour exporter. Claude te guide, surtout si tu n'as jamais touché à React.
Mon usage
Partie cruciale du Leo Reels Pipeline · Remotion prend les scripts vocaux ElevenLabs + les avatars Hedra + les visuels fal.ai et assemble des reels prêts à publier via Zernio. Un pipeline 1 clic = 5 reels programmés.

Ce qui n'est pas dans cette liste (volontairement)

J'utilise beaucoup d'autres outils dans mes projets Eurofiscalis et perso. Mais je n'ai pas envie de gonfler cette page pour faire plaisir au SEO. Voici ce que j'utilise aussi, mais que je ne te recommande pas de regarder tant que tu n'as pas maîtrisé les 6 ci-dessus :

  • Brevo · alternative à Resend avec CRM intégré. Plus puissant mais plus complexe.
  • TipTap · éditeur de texte riche pour formulaires. Seulement utile si tu construis une interface d'édition.
  • Stripe · paiements. Seulement utile si tu vends quelque chose.
  • Hedra / ElevenLabs · génération d'avatars vidéo et voix IA. J'en parle dans la section fal.ai / Remotion en tier 2 · ils fonctionnent souvent ensemble.
  • Graphify / Notion / Obsidian · gestion de notes et knowledge graph. Très persos, à choisir selon ton style.

Ma règle : ne jamais ajouter un outil à ta stack tant que tu n'as pas buté sur une limite réelle du précédent. 90 % des entrepreneurs que je vois ont trop d'outils et pas assez de clients. L'outil vient après le besoin.

Dashlane Lien affilié

Entre Claude Code, Supabase, Resend, Vercel, Stripe, Cloudflare, Sanity… je gère plus d'une vingtaine de clés API en permanence. Plus les mots de passe perso de la famille. Avant Dashlane, je les avais dans un fichier Notes, ou je cliquais sur « mot de passe oublié » cinq fois par semaine. Plus jamais.

Ce qu'il fait
Gestionnaire de mots de passe qui vit dans ton navigateur et sur ton téléphone. Il remplit tout seul, génère des mots de passe forts, stocke aussi les clés API, les codes 2FA, les documents sensibles. Chiffrement de bout en bout, personne chez Dashlane ne peut lire tes données.
Pour qui
Tous ceux qui ont plus de 10 comptes en ligne — donc tout le monde en 2026. Encore plus si tu utilises Claude Code et que tu jongles avec des clés API tous les jours.
Mon usage
Je l'ouvre 20 fois par jour sans y penser. Partage sécurisé activé avec Kevin pour les comptes pro Eurofiscalis. Zéro mot de passe réutilisé, zéro clé perdue. Un stress en moins.

— La dictée IA
Wispr Flow Lien affilié

Tu parles, ton texte s'écrit à l'écran — dans n'importe quelle app. Navigateur, Claude Code, mail, Notion. L'IA nettoie les hésitations, ponctue, reformule si tu bafouilles. Je dicte mes prompts Claude, je dicte mes drafts d'articles, je dicte mes réponses à la newsletter. Je tape trois fois moins qu'avant.

Ce qu'il fait
Appuie sur un raccourci, parle, relâche. Le texte apparaît là où tu avais le curseur. Fonctionne en français nickel. Tu peux même dicter des prompts longs à Claude Code sans jamais toucher au clavier.
Pour qui
Tous ceux qui écrivent beaucoup — articles, emails, prompts. Si tu tapes à deux doigts, encore plus. Si tu marches pendant tes appels et veux dicter des notes après, encore plus.
Mon usage
Je dicte mes idées de posts LinkedIn en allant chercher mon fils à l'école. Je dicte mes prompts Claude Code quand j'ai les mains dans un projet. Gain de temps énorme dès les premiers jours.
